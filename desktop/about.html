Sobre Mim

Meu nome é Gabriel Patricio, tenho 18 anos e sou estudante de Desenvolvimento de Software na Faculdade de Tecnologia (FATEC) de Osasco, sou apaixonado por tecnologias e por sua utilidade na vida das pessoas.

Decidi que a tecnologia faria parte da minha vida desde criança, sempre fui apaixonado por jogos e por toda a magia que a evolução desses meios trazem para a vida das pessoas. Dei meus primeiros passos como desenvolvedor na pandemia, quando ainda estava no Ensino Médio e participei de alguns cursos introdutórios sobre programação Front-End na Alura e na EBAC.

Atualmente, estou desenvolvendo meus conhecimentos na universidade no período noturno, na parte da manhã, faço um curso no SENAI de Implementação de Serviços em Nuvem da AWS Cloud Foundations (serviço de nuvem da Amazon),e na parte da tarde faço meus estudos na ALURA, focado mais em Front-End e Data Science, áreas que chamaram muito a minha atenção e pretendo me desenvolver nelas!

Hard Skills

Hard Skill é um conceito que pode ser definido como o conjunto de conhecimentos técnicos adquiridos com o tempo e a experiência.
Clique e veja algumas das habilidades que desenvolvi através dos meus estudos!


HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site.

JavaScript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.Trata a parte interativa e dinâmica da página.

SQL

SQL ou Structured Query Language (Linguagem de Consulta Estruturada) é uma linguagem padrão de gerenciamento de dados que interage com os principais bancos de dados baseados no modelo relacional.

AWS Cloud Foundations

AWS (Amazon Web Services) oferece soluções escaláveis para computação, armazenamento, bancos de dados, análises e muito mais.
